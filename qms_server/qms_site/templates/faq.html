{% trans 'General' %}

Q: {% trans 'What’s QuickMapServices?' %}
A: {% trans 'QuickMapServices (QMS) is an open catalog of geodata sources and a way to add them to your GIS in one click (read more).' %}

Q: {% trans 'What is the goal of QMS?' %}
A: {% trans 'To make geospatial services more accessible.' %}

Q: {% trans 'Why add services to QMS?' %}
A: {% trans 'To easily find them and plug-in to your GIS project. Literally in just few clicks.' %}

Q: {% trans 'Can I add services to GIS without QMS?' %}
A: {% trans 'Sure! But you’ll have to enter cumbersome links and parameters.' %}

{% trans 'Using a service' %}

Q: {% trans 'How do I use one of the services from QMS?' %}
A: {% trans 'Search and add it in your GIS. Currently there are clients for QGIS, ArcGIS and NextGIS Mobile.' %}

Q: {% trans 'What kind of data I will get by using one of QMS services?' %}
A: {% trans 'This depends on a service. Through QMS you can get access to vector data, basemaps and rasters.' %}

{% trans 'Adding a service' %}

Q: {% trans 'What kind of services I can add to QMS?' %}
A: {% trans 'You can add:' %}

  • TMS – Tile Map Service.
  • WMS – Web Map Service.
  • WFS – Web Feature Service.
  • {% trans 'GeoJSON – geodata in GeoJSON format. Link to file or data stream.' %}

Q: {% trans 'What are minimum needs to add a service?' %}
A: {% trans 'You’ll need to provide a correct service url at the very least.' %}

Q: {% trans 'All these abbreviations mean absolutely nothing to me, can I still add a service?' %}
A: {% trans 'Unfortunately, not at this point, you’ll need to learn the basics first.' %}

Q: {% trans 'Ok. I’ve found a website with a map a like and copied it’s link, can I use it?' %}
A: {% trans 'Most probably not, links copied from browsers address bar are typically not service links that you can use with QMS.' %}

Q: {% trans 'What do correct service links look like?' %}
A: {% trans 'Here are few examples and their particular features:' %}

  • TMS – http://tile.openstreetmap.org/{z}/{x}/{y}.png
    • {% trans 'This service provides tiled images' %}
    • {% trans 'Note z, x, y – those are tile numbers, they correspond to a particular area' %}
  • WMS – http://cartoweb.cma.gva.es/arcgis/services/cartografia_tematica/MapServer/WMSServer
    • {% trans 'This service provides untiled-imagery' %}
    • {% trans 'Note words like ‘wms’ or ‘wmsserver’ in the link' %}
  • WFS – http://www.wfs.nrw.de/geobasis/wfs_nw_alkis_vereinfacht?REQUEST=GetCapabilities&SERVICE=WFS
    • {% trans 'This service provides vector data' %}
    • {% trans 'Note words like ‘wfs’ or WFSserver’ in the link' %}
  • GeoJSON – http://opengeo.intetics.com.ua/osm/pa/data/protected_area_polygon.geojson
    • {% trans 'This service provides vector data and is probably the simplest, being just a file with data' %}
    • {% trans 'File can be hosted anywhere, as long as it’s valid GeoJSON' %}

Q: {% trans 'I’ve seen services links that are different from examples above, are there other examples?' %}
A: {% trans 'Yes, there are lots of varieties and service link can look very differently. Just browse QMS clicking on service type and see what kind of links other services show.' %}

{% trans 'Start with the ones down below.' %}

Q: {% trans 'I’ve created a service, but type in incorrect, can I change it?' %}
A: {% trans "You can't change the type of a service once created. You'll need to create another one with the correct type." %}

Q: {% trans 'How can I delete a service?' %}
A: {% trans "You can't delete services at the moment. Report a problem and we'll remove it soon." %}

Q: {% trans 'How do I check that my service is OK?' %}
A: {% trans 'Find it through a client and add to see the data on a map. In QGIS you can use QuickMapServices plugin.' %}

{% trans 'Fixing a service' %}

Q: {% trans 'I’ve found service, but it doesn’t work, how do I fix it?' %}
A: {% trans 'Use ‘report a problem’ form.' %}

{% trans 'Every service has it’s author (maintainer, submitter). Use the form to let the submitter know that the service is broken. Provide information how to fix it.' %}

Q: {% trans 'I’ve notified service maintainer about problem and provided solution, but maintainer is not reacting, what can I do?' %}
A: {% trans 'If maintainer doesn’t reply for a long time – create a new service (you’ll become it’s maintainer). Old incorrect service will gradually be removed.' %}
